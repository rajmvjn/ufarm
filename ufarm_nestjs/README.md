## Description

Farmer App micro service built using Nestjs framework. It will contain API's for product selling,purchase, user account, checkout and farming guidelines. MongoDB acts as backend service for database activities.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
